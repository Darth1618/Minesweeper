Detailed Description

Класс игрового поля сапёра.

Хранит два слоя: реальное поле (map) и то, что видит игрок (mask). Обеспечивает логику открытия ячеек, расстановки мин и флагов.

◆ openCell()

bool Map::openCell ( int x,
int y )
inline

Открывает ячейку по клику игрока.

Если на ячейке стоит флаг — игнорирует клик. Если под ячейкой мина — возвращает false (проигрыш). Иначе рекурсивно открывает ячейку и соседей через openEmpty().

Parameters
xКоордината X ячейки в массиве поля.
yКоордината Y ячейки в массиве поля.
Returns
true — ячейка успешно открыта, игра продолжается.
false — под ячейкой мина, игра окончена.
