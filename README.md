# Сапёр (Minesweeper)

Классическая игра «Сапёр» на C++ с использованием библиотеки SFML 3.

## Возможности

- Три уровня сложности: Юниор (9×9, 10 мин), Любитель (16×16, 40 мин), Профи (22×22, 99 мин)
- Первый клик всегда безопасен — мины расставляются после первого хода
- Система флагов (правая кнопка мыши — пометить подозрительную ячейку)
- Таймер для каждой игровой сессии
- Автоматический возврат в меню после победы/поражения (через 5 секунд)

## Требования

- C++20
- [SFML 3.0.2](https://www.sfml-dev.org/)
- CMake 3.20+
- Файл шрифта `arial.ttf` в корне проекта

## Сборка и запуск

```bash
mkdir build && cd build
cmake ..
cmake --build .
./Minesweeper
```

Или открой проект в **CLion** и нажми **Run**.

## Структура проекта

```
Minesweeper/
├── main.cpp          # Игровой цикл, отрисовка, обработка ввода
├── map.h             # Класс Map — игровая логика (мины, ячейки, flood fill)
├── CMakeLists.txt
├── arial.ttf
├── tests/
│   ├── doctest.h     # Фреймворк для тестирования (header-only)
│   └── test_map.cpp  # Юнит-тесты для класса Map
└── docs/             # Документация, сгенерированная Doxygen
```

## Запуск тестов

Выбери конфигурацию `minesweeper_tests` в CLion и нажми **Run**, или через терминал:

```bash
cd cmake-build-debug
./minesweeper_tests.exe
```

## Управление

| Действие | Управление |
|----------|-----------|
| Открыть ячейку | Левая кнопка мыши |
| Поставить / убрать флаг | Правая кнопка мыши |
| Вернуться в меню | Клик в любом месте (на экране победы/поражения) |
